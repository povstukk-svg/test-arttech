# ArtTech Bot

## Что умеет бот

* команда `/start` показывает приветствие и меню;
* команда `/help` показывает помощь;
* кнопка «О нас» выводит информацию о компании;
* кнопка «Каталог» показывает категории товаров;
* кнопка «Связаться» показывает контакты;
* бот отвечает на обычные текстовые сообщения.

## Технологии

* Python
* python-telegram-bot
* python-dotenv

## Структура проекта

```text
arttech bot/
│
├── bot.py
├── requirements.txt
├── .env.ex
└── README.md
```

## Запуск

На Windows запустить проект через `run.bat`.
## Проверка

Открыть бота в Telegram @yosucooobot и отправить команду:

```text
/start
```

После этого бот покажет меню с кнопками.




# ArtTech Landing

## Что реализовано

- шапка с названием и навигацией;
- Hero-блок с заголовком, текстом и кнопкой;
- блок преимуществ с 4 карточками;
- секция контактов;
- футер;
- адаптивная верстка для телефона;
- hover-эффекты и простая анимация.

## Технологии

- HTML
- CSS

## Структура проекта

```text
arttech landing/
│
├── index.html
├── style.css
└── README.md
```

## Как запустить

Открыть файл `index.html` в браузере.

Также можно запустить через Live Server в VS Code.
